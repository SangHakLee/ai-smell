# AI Smell Detector

A tool to detect if a website is potentially generated by AI builders (Lovable, v0.dev, Bolt, etc.) or uses generic, low-effort templates.

## How it works

It fetches a URL and runs a series of "sniffers" to analyze the page. Each sniffer looks for specific AI patterns:

- **Domain (üéØ Strongest):** Detects AI service default domains (lovable.app, v0.dev, bolt.new, etc.)
  - See configurable list in `src/sniffers/domain.ts`
- **Meta:** Inspects meta tags for AI builders (Lovable, v0, Cursor) and AI-generated descriptions
- **Boilerplate:** Detects React/Vite boilerplate, empty SPA skeletons, generic favicons
- **Comments:** Finds TODO comments and AI placeholder text in production code
- **ColorPalette:** Identifies AI-favorite colors (#0061FF, #0052D9, vibrant oranges, purple gradients)
  - Analyzes both inline styles and external CSS files
- **Content:** Analyzes text for AI-written marketing phrases
- **Design:** Looks for uncustomized layout patterns
- **UI Kit:** Checks for default themes from popular UI kits

The results are weighted and aggregated into a final "smell score".

## Installation

```bash
npm install
```

## Usage

```bash
npm run dev -- <URL>
# or after building
npm run build
npm start -- <URL>
```

## Adding New AI Service Domains

You can easily add new AI service domains by editing the constants in `src/sniffers/domain.ts`:

```typescript
export const AI_SERVICE_DOMAINS = [
  'lovable.app',
  'v0.dev',
  'bolt.new',
  // Add your domains here
];

export const HIGH_CONFIDENCE_AI_DOMAINS = [
  'lovable.app',    // Almost certainly AI-generated
  'v0.dev',
  // Add high-confidence domains here
];
```

Current AI service domains detected:
- **High-confidence AI builders:** Lovable, v0.dev, Bolt.new, GPT Engineer, Replit
- **Popular deployment platforms:** Vercel, Netlify, GitHub Pages, Cloudflare Pages
- **No-code platforms:** Webflow, Wix, Squarespace, Bubble, Framer, Carrd
- See full list in `src/sniffers/domain.ts`
